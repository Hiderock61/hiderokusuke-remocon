# ヒデロクスケ・リモコン v0.2

5つ以上の役割へ戻るための、個人用コックピット。  
押して、コピーして、ひでGPTや担当AIへ渡すための静的 Web アプリ。

必ず書くこと（重要）

- これは一般向け AI リモコンではなく、ヒデロック OS の個人用コックピットである。
- ai-remocon と混ぜないこと。
- ai-remocon は比較的一般化された AI リモコンであるが、hiderokusuke-remocon は個人用である。
- hiderokusuke-remocon は、ひでGPT、明智君、体感、現実君、燃料線、夢ベクトルなどへ戻る個人用リモコンである。
- ボタンを押すと、その役割に渡すプロンプトが表示される。
- 表示された文をコピーして ChatGPT / Gemini / Copilot / Claude などへ貼ることを想定している。

主な役割（例）

- ひでGPT
- 明智君
- 体感
- 現実君
- 燃料線
- 夢ベクトル
- 停止
- ヤスリ
