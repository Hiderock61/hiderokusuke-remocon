# フリーランス装備庫 v2.0

フリーランス装備庫は、AI時代の仕事道具を全部覚えるためのアプリではありません。

Slack、GitHub、Figma、Canvaなどの道具を、iPhoneだけで「見学する」「今日の一手を選ぶ」「相手に渡す形にする」ための訓練アプリです。

v2.0では、納品タブに **3つのお渡しレターモード** を追加しました。

## 公開URL

https://hiderock61.github.io/freelance-/

## v2.0で追加したこと

納品タブの「完成文を見る」側で、提出文の種類を3つから選べるようにしました。

- 📦 **初回提出**：初めて成果物を渡す時の文章
- 🛠 **修正提出**：修正後に再提出する時の文章
- 🚧 **途中共有**：完成前に方向性を確認してもらう時の文章

入力欄はv1.0のまま増やさず、出力文だけを状況に合わせて切り替えます。

## 6つのタブの役割

- 👁 **見学**：仕事の1日の流れを見る
- 🔑 **共有**：URL・権限・コメントの感覚を知る
- 🧰 **装備**：道具ごとの距離感を見る
- 📍 **今日**：今日の3ステップを選ぶ
- 📦 **納品**：提出文を作る
- 🕵️ **監督**：AI出力をそのまま貼らないために確認する

## v2.0であえて入れていないもの

- 保存機能
- 画像アップロード
- PDF出力
- ログイン機能
- 新規タブ追加
- 入力欄追加

## 実機確認ポイント

- トップが v2.0 表記になっている
- 納品タブに3つのレター切り替えボタンが出る
- 初回提出 / 修正提出 / 途中共有でプレビューが切り替わる
- コピー内容が選択中のテンプレと一致する
- iPhoneで横スクロールが出ない
- 見学 / 共有 / 装備 / 今日 / 監督 が壊れていない
